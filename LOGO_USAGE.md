# setfn 로고 활용 가이드

이 문서는 `assets/logo/setfn.png` 로고 파일을 웹에 안전하고 일관되게 사용하는 방법을 정리합니다.

요약
- 경로: `assets/logo/setfn.png`
- 권장 크기: 사용처에 따라 32/48/96/128px 등. (아이콘용 32~48px, 헤더/영웅 배너 96~128px)
- 고해상도(레티나): 배율 2x 이미지를 준비하면 선명합니다 (`setfn@2x.png`).
- 형식: PNG(투명 배경) 또는 SVG(벡터 권장)

웹 사용 예시
- HTML 인라인 이미지
  <img src="/assets/logo/setfn.png" alt="setfn" width="96" height="96">

- 링크로 홈 연결
  <a href="/"> <img src="/assets/logo/setfn.png" alt="setfn"> </a>

접근성
- `alt` 속성은 의미 있는 텍스트 (예: "setfn")로 설정하세요.
- 단순히 장식용이면 `alt=""`로 두어 스크린리더가 무시하게 하세요.

파비콘
- 작은 아이콘(16/32px)은 별도 `favicon.ico` 또는 `favicon-32x32.png`로 준비하세요.

색상/배치
- 로고가 컬러일 경우 어두운 배경 위에 놓으면 대비가 좋습니다.
- 배경이 어두울 때 로고에 흰색 경계나 반투명 패널을 넣어 가독성을 확보하세요.

레이아웃 팁 (유사 디자인 참고)
- 전체 페이지 중앙에 큰 로고 + 간결한 메시지를 배치하면 준비중 페이지에서 효과적입니다.
- 오른쪽 상단 등 작은 영역에 로고를 두고 텍스트를 왼쪽에 배치하는 레이아웃도 깔끔합니다.

파일 관리
- 원본(가능하면 SVG)을 `assets/logo/setfn.svg`로 보관하세요.
- 래스터 버전(PNG)은 필요 크기별로 `assets/logo/setfn-96.png`처럼 관리하면 편합니다.

추가 작업
- 원하시면 SVG 변환(벡터화) 또는 2x/아이콘용 파일을 생성해 드릴게요.
